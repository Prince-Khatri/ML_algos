#  Linear Regression

### How to Start

#### 1. Create and activate a virtual environment
```bash
python3.10 -m venv env
source env/bin/activate
```

#### 2. Run the preprocessing module (from project root)
```bash
python3 -m Linear_Regression.preprocess
```

#### 3. Train the model
```bash
python3 -m Linear_Regression.train
```

---

**Congratulations!**  
You have successfully trained a machine learning model using a linear regression algorithm implemented entirely from scratch.
