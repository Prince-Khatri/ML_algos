# ML from scratch
This is a repo which implements all the basic machine learning algorithms from scratch.
## Currently Building :-
1. Linear Regression on Boston_dataset

## Upcoming :-
1. Multiple Regression
2. Logistic Regression

#### Steps to get started :- 

1. Install env
```
python3 -m venv env
source env/bin/activate
```

2. Install requirements.txt

```
pip install -r requirements.txt
```
